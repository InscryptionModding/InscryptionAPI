    Class TribeManager

    This class handles the addition of new tribes into the game

    Inheritance

    Syntax
    [HarmonyPatch]
public static class TribeManager
    Remarks

    This manager can currently handle watermarking cards with tribes and having them appear at tribal choice nodes. Totems are not currently supported.
