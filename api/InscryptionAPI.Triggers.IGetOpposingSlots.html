    Remarks

    If your card is replacing the default attack slot (the opposing slot) see 'RemoveDefaultAttackSlot' If you are not replacing the default attack slot, do not include that here. Simple ensure 'RemoveDefaultAttackSlot' returns false. Only return the default attack slot if you want to attack it an additional time.

    RemoveDefaultAttackSlot()

    If true, this means that the attack slots provided by GetOpposingSlots should override the default attack slot. If false, it means they should be in addition to the default opposing slot.
